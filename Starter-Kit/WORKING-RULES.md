# Working Rules / Reglas de Trabajo

> **Assistant: read this file AND `PROJECT-STATE.md` at the start of every session.**
> **Asistente: lee este archivo Y `PROJECT-STATE.md` al inicio de cada sesión.**

## ES
1. **Todo bilingüe** (inglés y español). El sitio arranca en **inglés** por defecto.
2. **Mantén el estilo y la estructura** del proyecto (index.html, videos.json, animations/, assets/, tools/).
3. **Pregunta antes de editar** un archivo existente.
4. **Versiona antes de reemplazar:** copia el archivo actual a `Versiones/` como `nombre-vN-AAAA-MM-DD.ext` y luego aplica los cambios.
5. **Módulos:** usa el motor de `animations/module-template.html` (header fijo, navegación por pasos, tema claro/oscuro, animaciones en **canvas**). Fotos reales en `assets/` (JPG, nombres en inglés), referenciadas como `../assets/…`.
6. **Diseño responsivo obligatorio** (PC y móvil): sin scroll horizontal, botones tocables, textos legibles.
7. **Mantén `PROJECT-STATE.md` actualizado** en cada mejora o cambio relevante (con su fecha).

## EN
1. **Everything bilingual** (English and Spanish). The site starts in **English** by default.
2. **Keep the project style and structure** (index.html, videos.json, animations/, assets/, tools/).
3. **Ask before editing** an existing file.
4. **Version before replacing:** copy the current file to `Versiones/` as `name-vN-YYYY-MM-DD.ext`, then apply changes.
5. **Modules:** use the `animations/module-template.html` engine (sticky header, step nav, light/dark theme, **canvas** animations). Real photos in `assets/` (JPG, English names), referenced as `../assets/…`.
6. **Responsive is mandatory** (desktop and phone): no horizontal scroll, tappable buttons, readable text.
7. **Keep `PROJECT-STATE.md` up to date** on every meaningful improvement or change (with its date).
