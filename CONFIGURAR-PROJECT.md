# Cómo crear el Project en Claude / How to set up the Claude Project

Guía rápida para dejar todo listo. *Quick guide to get everything ready.*

---

## Paso 1 · Crear el Project / Create the Project

En la app de Claude, barra lateral izquierda: **Projects → + New project**.
*In the Claude app, left sidebar: **Projects → + New project**.*

**Nombre / Name:**
```
Videos de Entrenamiento / Training Videos
```

---

## Paso 2 · "What are you trying to achieve?"

Claude te pedirá esto al crear el Project. Copia y pega esta respuesta:
*Claude asks this when you create the Project. Copy and paste this answer:*

```
ES: Quiero un portal de capacitación bilingüe (español e inglés) para mi equipo,
publicado en GitHub Pages, donde mis trabajadores —tanto los nuevos (onboarding)
como los actuales— entren, vean videos y animaciones cortas de entrenamiento en
su propio idioma y se capaciten solos. Los clips los creo combinando código
(HTML/CSS/JS) y diseño. Al mismo tiempo, el sitio sirve como muestra profesional
de mi trabajo y como biblioteca organizada de todos los videos que produzco.
Tendré éxito cuando el sitio: (1) se vea profesional, (2) sea fácil agregar
nuevos videos sin programar, (3) funcione perfecto en ambos idiomas, y (4) esté
publicado con un link vivo que cualquiera del equipo pueda abrir.

EN: I want a bilingual (Spanish and English) training portal for my team,
published on GitHub Pages, where my workers —both new hires (onboarding) and
current staff— can log in, watch short training videos and animations in their
own language, and train themselves. I create the clips by combining code
(HTML/CSS/JS) and design. At the same time, the site works as a professional
showcase of my work and as an organized library of every video I produce.
I'll know it works when the site: (1) looks professional, (2) makes it easy to
add new videos without coding, (3) works flawlessly in both languages, and (4)
is published with a live link anyone on the team can open.
```

---

## Paso 3 · Instrucciones del Project / Project instructions

Pega también este texto en el campo de instrucciones (custom instructions):
*Also paste this text into the instructions field (custom instructions):*

```
ES: Ayúdame a crear videos y animaciones cortas de entrenamiento combinando
código (HTML/CSS/JS) y diseño. TODO el contenido debe ser bilingüe: español e
inglés. Mantén el mismo estilo visual y la estructura de carpetas del proyecto
(index.html, videos.json, videos/, animations/, assets/). El resultado se
publica en GitHub Pages.

EN: Help me create short training videos and animations combining code
(HTML/CSS/JS) and design. ALL content must be bilingual: Spanish and English.
Keep the project's visual style and folder structure (index.html, videos.json,
videos/, animations/, assets/). The result is published on GitHub Pages.
```

---

## Paso 4 · La carpeta / The folder

Esta carpeta (`Project`) ya está dentro de tu Google Drive, así que se sincroniza sola.
*This folder (`Project`) is already inside your Google Drive, so it syncs automatically.*

En el Project de Claude, adjunta esta carpeta de Drive como fuente de contexto.
*In the Claude Project, attach this Drive folder as a context source.*

---

## Paso 5 · Publicar / Publish

Sigue la sección **"Publicar en GitHub Pages"** del `README.md`.
*Follow the **"Publish to GitHub Pages"** section in `README.md`.*
